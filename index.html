Game types

Fixed range

This game mode lets you set a number interval to guess in at the beginning that is then kept for as long as you want to play.

Fluid range

This game mode lets you change the number interval to guess in before every guess.

Random range

This game mode give you a random interval your number can be inside for every guess.

About this project

This was initially a small assignment in my first course at the JavaScript developer front-end program at Lernia YH. The first assignment was done with alert boxes and you can still find it as the first release on Github. I decided to use the project to get some more practice in CSS and design by giving it a proper home here. This was also meant to give me some idea for creating my portfolio website.
